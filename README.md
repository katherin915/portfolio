# Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my skills, projects, certifications, achievements, and coding profiles.

---

## Features

- Responsive modern UI design
- Dark / Light mode toggle
- Animated particles background
- Typing text animation
- Responsive navigation bar
- Downloadable resume
- Project showcase section
- Certifications gallery with images
- Social media links
- Mobile-friendly layout
- Smooth scrolling and hover animations

---

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Particles.js

---

## Sections Included

- Home
- About
- Skills
- Education
- Projects
- Certifications
- Achievements
- Coding Profiles
- Contact

## Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Open the project folder

```bash
cd portfolio
```

3. Run the project

Open `index.html` in your browser  
OR use VS Code Live Server extension.
